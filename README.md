# 🤖 AI-Assisted Web App — Text Tools

A collection of smart text utility tools built with AI assistance.

## Features
- **Word Counter** — counts words, characters, and sentences in real-time
- **Case Converter** — UPPERCASE, lowercase, Title Case, Sentence case
- **Text Reverser** — reverse by words, characters, or lines
- **Password Generator** — customizable strong password generator

## Technologies
- HTML5
- CSS3
- Vanilla JavaScript
- Built with AI tools assistance

## How to Use
Open `index.html` in any browser — no installation needed.
